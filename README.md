## Instrucciones para instalar ESLint

```
npm install eslint --save-dev
npm init @eslint/config
```

### Reglas a probar


## Instrucciones para instalar Prettier

```
npm install --save-dev --save-exact prettier
echo {}> .prettierrc.json
```
